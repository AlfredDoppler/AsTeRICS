CSV Reader

Component Type: Sensor (Subcategory: File System)

This plugin reads an .csv file and sends a string to the output.


ReadCSV

Event Listener Description

  • read: Opens a menu to chose a file which should be opened.

Output port Description

  • Output[string]: The port send the data as a string. An example for the string is: "data1;data2;data3". To seperate the string the StringSplitter can be used.

Properties

  • FilePath[string]: Defines the Path were the file is located.
