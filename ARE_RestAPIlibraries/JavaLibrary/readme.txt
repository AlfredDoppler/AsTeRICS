=======================================
Java Client for ARE rest API
=======================================

Recommended IDE: 
eclipse

Recommended java version:
jdk7

Instructions:
1) Create a simple java project
2) Copy the 'src', 'lib', 'models' folder to the root of your project
3) Add the jar files inside 'lib' folder to the projects' build path
4) Run 'JavaClient.java' or write your own tester class
